PRODUCTION MAJOR PROJECT

Student Internship Hub (SIH)

EdTech + Recruitment Platform

Problem Statement

Students struggle to find genuine internships due to fake postings and lack of transparency. Companies cannot efficiently find verified candidates. No unified platform existed to connect students, companies, and educational institutions with proper verification and tracking.

Solution Overview

Built a centralized internship management platform with three distinct user roles. Students browse verified internships and track applications in real-time. Companies post opportunities and manage hiring pipelines. Administrators ensure platform integrity through verification and moderation workflows.

Core Features

  • JWT-based authentication with role-based access control (Student, Company, Admin)
  • Real-time application status tracking and notifications
  • Company verification system with admin approval workflow
  • Resume upload and profile management for students
  • Admin dashboard for content moderation and user verification
  • Application analytics and hiring pipeline management

System Architecture

RESTful API architecture with separated frontend and backend layers. Implements JWT authentication, role-based middleware, and normalized MySQL database with proper foreign key relationships. Built with Express.js routing, input validation, and error handling middleware for production stability.

Tech Stack

Node.js Express MySQL JWT Auth RESTful API Railway HTML5 CSS3 JavaScript

Engineering Learnings

  • Resolved race conditions in concurrent application submissions
  • Optimized database queries with proper indexing and joins
  • Implemented secure file upload handling for resumes
  • Debugged authentication edge cases in production environment
  • Deployed and monitored application on Railway platform
