### JavaScript Animation Engine:
Completely rewritten with **class-based architecture**:

1. **AnimationEngine** (Central Controller):
   - `requestAnimationFrame` loop for smooth 60fps
   - Scroll tracking with debouncing
   - Parallax effects on grid background
   - Active nav link detection
   - Metric counter animations

2. **TextScrambler**:
   - Character-by-character reveal
   - Random character scrambling effect
   - Used on section titles with `data-scramble`

3. **MagneticButton**:
   - Mouse-follow effect on buttons
   - Subtle magnetic pull (20px strength)
   - Smooth transform transitions

4. **SkillProgressAnimator**:
   - Spring easing for progress bars
   - IntersectionObserver triggers
   - 1500ms duration with cubic easing

5. **IntersectionObserver System**:
   - Reveal animations (fade + slide)
   - Timeline stagger (150ms delay)
   - Project card stagger (100ms delay)
   - Threshold-based triggering
